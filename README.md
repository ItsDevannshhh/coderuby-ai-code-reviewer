# CodeRuby – AI Code Reviewer

An AI-powered GitHub code review platform that automatically reviews pull requests, provides actionable feedback, and helps developers improve code quality using Large Language Models.

## ✨ Features

* 🔐 GitHub Authentication using Better Auth
* 📦 GitHub App integration
* 🤖 AI-powered pull request reviews
* 💬 Inline review comments and suggestions
* 📂 Repository management
* ⚡ Background job processing with Inngest
* 🎨 Modern UI built with Next.js, React, Tailwind CSS & shadcn/ui
* 🔒 Secure authentication and session management

## 🛠 Tech Stack

### Frontend

* Next.js 15 (App Router)
* React
* TypeScript
* Tailwind CSS
* shadcn/ui

### Backend

* Next.js Server Actions
* Better Auth
* PostgreSQL
* Prisma ORM
* Inngest

### Integrations

* GitHub OAuth
* GitHub App
* AI Models
